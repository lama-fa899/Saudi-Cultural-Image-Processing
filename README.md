# Saudi-Cultural-Image-Processing

This is a Python project in which I built an image processing system to analyze and classify Saudi cultural objects using computer vision techniques.

I preprocessed the images by applying different noise models, intensity transformations, image restoration filters, morphological operations, feature extraction, and PCA analysis. I then evaluated the performance of different restoration methods using MSE and PSNR metrics to identify the most effective filtering techniques for each type of noise.

Libraries used:

- OpenCV

- NumPy

- Pandas

- Matplotlib

- Scikit-learn

- Scikit-image

- SciPy

- Gradio
